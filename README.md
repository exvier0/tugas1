# UTS Pemrosmt5 - Aplikasi Daftar Mahasiswa

Proyek Flutter sederhana untuk menampilkan data mahasiswa menggunakan ListView.

Struktur penting:
- `lib/models/` - model dan data sample
- `lib/screens/` - layar daftar dan detail
- `lib/widgets/` - komponen UI dapat digunakan ulang
- `assets/images/` - aset gambar

Cara menjalankan:
```powershell
cd "c:\Users\E L I T E B O O K\Documents\flutter\uts_pemrosmt5"
flutter pub get
flutter run
```
